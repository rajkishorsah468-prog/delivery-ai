# 🚀 AI Delivery Environment

This project simulates a real-world delivery system using Reinforcement Learning.

## 🔧 Features
- Custom OpenAI-style environment
- Q-learning agent
- Reward-based learning
- Training visualization (graph)

## 📁 Files
- env.py → environment logic
- train.py → training agent
- render.py → visualization

## ▶️ How to Run
pip install numpy matplotlib  
python train.py  

## 📊 Output
Graph showing learning progress of the AI agent.

## 💡 Use Case
- Logistics optimization  
- Path planning  
- AI-based delivery systems
